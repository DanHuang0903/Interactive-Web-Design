Inteactive Web Desgin

Interactive web desgin is the most important aspect of UX (user experience). Function-wisely, interaction with web pages has been sophisticated for years. The reason that interactive web design is still a hot topic is about "design". It's about how we visualize the data, and how we have the interaction more diverting. And that is the reason we need to introduce more powerful programming languages, libaray, and frameworks, which I am familar with.
