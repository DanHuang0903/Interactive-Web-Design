Inteactive Web Desgin

Interactive design is regarded as one of the most important aspects of UX (user experience) for online web users. With all the dynamic tools and scripts functioning well, the key to interactive web design is about the "Design". It's not only about making the scripts functional, but also about how to bring the interactions to live, how to visualize the data, and how to elevate the UX to another level. For students of SISLT, we can accomplish such instructional goals through an updated Interactive Web Design course introducing more powerful programming languages, library, and frameworks, and most importantly, where and how to use them.
